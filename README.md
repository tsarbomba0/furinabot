# Furina
A bot early in development named after Furina de Fontaine

## Functions

- Rule34 API wrapper
- Music player bot!
- Websocket for song information!

## Requirements

- node.js
- lavalink-client
- discord.js
- working lavalink node (provided configuration in lavalink/ let's you self-host it!)
